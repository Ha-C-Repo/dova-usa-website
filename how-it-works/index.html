Eight Steps · End to End

How DOVA Works.

From installation to sale - eight steps, every event on the record.

Download PDF Version Book a Demo
01

Install

Every U.S. vehicle since 1996 has an OBD-II port - federally mandated, physically identical across all makes and models. A DOVA module plugs into that port. Installation takes under one hour. No OEM permission required. No special tools.

02

Auto-Enroll

At power-on, the module reads the vehicle's VIN directly from the CAN bus and transmits it to the DOVA cloud. The vehicle is automatically enrolled to the fleet account. No technician configuration. No make/model programming. Live in under 60 seconds.

03

Fleet Setup

Your administrator configures the web dashboard - assign roles, set permissions per user and vehicle, define geofence boundaries. Each authorized user sees only the vehicles they're cleared to access.

04

Access Request

An authorized user opens the DOVA mobile app, selects the target vehicle, and submits an access request. That request travels to the DOVA cloud authorization engine - never directly to the hardware module.

05

Three-Gate Check

The cloud authorization engine runs three sequential checks: (1) the requesting user's identity is verified via two-factor authentication; (2) the user's role is confirmed as authorized for that vehicle; (3) if configured, affirmative approval is received from a designated manager. All three must pass, in order.

06

Vehicle Unlocked

Only after all three gates clear does the cloud send a cryptographically signed command via 4G LTE / 5G cellular to the hardware module, which triggers the door lock or unlock actuator. No shortcut. No bypass. No exceptions.

07

Every Access Logged

The event is immediately written to a permanent, tamper-evident log: who accessed the vehicle, when, GPS coordinates, and a real-time OBD-II diagnostic snapshot - odometer, fuel, battery state of charge, tire pressure, and active fault codes. Cryptographically bound. Cannot be modified or deleted.

08

Vehicle Sells

When the vehicle is sold or transferred, you choose what happens to the DOVA module. See Post-Sale Options for the complete set of choices - for your business and for your customer.
