About this prototype

  • This is the Phase 1.5 prototype. The production site is at dovausa.com.
  • Cinematic hero stills generated with Google AI Studio (Nano Banana / Imagen).
  • Scroll-driven crossfade and zoom orchestration via GSAP ScrollTrigger.
  • Smooth-scroll inertia via Lenis.
  • Particle layer above imagery is hand-drawn 2D canvas (no Three.js dependency, fast load).
  • Mobile-responsive: stacks to a single column, image first.
  • Phase 2 (production) would add a real 3D vehicle wireframe model and a contracted 3D artist for bespoke assets.
