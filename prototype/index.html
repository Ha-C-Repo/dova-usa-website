Prototype Notes

What this prototype is.

This is a Phase 1 prototype, not production. It shows a 3D interactive hero replacing the static one on the live site. The scene above renders in real-time WebGL via Three.js: a stylized OBD-II port with a DOVA module installed, particle data flowing out to a glowing cloud sphere representing the authorization engine.

Interactions: the scene slowly auto-rotates and responds to mouse movement (parallax). The data particles loop continuously. On mobile, the scene scales down but remains interactive.

Built in one session using hand-coded Three.js geometry primitives (no external 3D model files). Phase 2 production would replace the primitives with hand-modeled GLB assets, add the scroll-driven How-It-Works narrative, and ship to a contract Three.js developer for polish and optimization.
