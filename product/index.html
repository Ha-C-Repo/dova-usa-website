The DOVA Platform

Three integrated layers. One seamless system. No OEM agreements required.

Layer 1 Patent Pending

Purpose-Built for Every Vehicle Since 1996

A cellular telematics module installed via the vehicle's OBD-II diagnostic port. Powered by DOVA proprietary firmware. Encrypted end-to-end with AES-256. Compatible with every U.S. OBD-II vehicle — without make/model-specific programming.

  • 4G LTE / 5G cellular radio — sole command-receiving channel. No Bluetooth. No NFC.
  • Embedded GNSS receiver — real-time GPS coordinates
  • Vehicle CAN bus interface — reads OBD-II PIDs, triggers door lock/unlock actuators
  • Relay output fallback — covers 100% of OBD-II vehicles
  • VIN verification at every power cycle
  • Tamper detection — GPS-stamped alert transmitted before shutdown
  • AES-256 encrypted communication
  • Universal firmware — no make/model-specific builds required
Layer 2 Core Patent Asset

Three Gates. In Sequence. No Exceptions.

The brain of the DOVA platform. A stateful sequential authorization pipeline that physically cannot dispatch a vehicle command until three mandatory conditions are satisfied in order.

Gate 1

Two-Factor Authentication

The requesting user's identity is verified with a second factor before any process continues.

Gate 2

Role-Based Permission

The user's role is evaluated in real time against the specific vehicle and command type requested.

Gate 3

Manager Approval

Where configured, an affirmative approval from a designated manager-role account is required before the command is dispatched.

Additional Cloud Features

  • Geofence enforcement — server-side, administrator-defined zones
  • Tamper-evident access log — write-once, append-only, cryptographically timestamped
  • Diagnostic snapshot binding — OBD-II data cryptographically tied to each access transaction
  • Real-time GPS fleet map
  • Role & permission management system
  • Alert engine — access events, geofence violations, tamper alerts, diagnostic thresholds
Layer 3

Simple Interfaces. Powerful Control.

Mobile and web applications built for the people who actually run the fleet — drivers, managers, administrators.

Mobile App (iOS + Android)

  • Permission-scoped vehicle list — users see only their authorized vehicles
  • One-tap access request
  • Real-time authorization status — shows pending manager approval where required
  • Instant access confirmation with GPS-verified vehicle location
  • Vehicle diagnostic display
  • Push notifications for approvals, denials, and tamper alerts

Web Dashboard

  • Real-time fleet map — live GPS positions, all enrolled vehicles
  • Full access log — filterable by user, vehicle, and date range
  • CSV / PDF export for compliance reporting
  • User management — add, remove, role assignment
  • Geofence builder — create and assign geographic access zones
  • Alert configuration — access events, violations, diagnostic thresholds
